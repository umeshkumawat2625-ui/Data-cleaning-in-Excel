# 📊 Data Cleaning in Excel Project

## 🚀 Overview

This project demonstrates the process of cleaning and preparing raw data using Microsoft Excel. Real-world datasets often contain missing values, duplicate records, inconsistent formats, and data quality issues that must be addressed before analysis.

The objective of this project is to transform messy data into a clean, structured, and analysis-ready dataset using Excel's built-in data cleaning features.

---

# 🎯 Project Objectives

* Clean raw datasets using Microsoft Excel.
* Remove duplicate records.
* Handle missing values.
* Standardize data formats.
* Improve overall data quality.
* Prepare data for analysis and reporting.

# ⚙️ Data Cleaning Process

## Step 1: Data Inspection

Review the dataset to identify:

* Missing values
* Duplicate records
* Incorrect formats
* Inconsistent text values

---

## Step 2: Remove Duplicate Records

Used Excel's **Remove Duplicates** feature to eliminate redundant entries.

### Benefits

* Improved data quality
* Accurate reporting
* Reduced redundancy

---

## Step 3: Handle Missing Values

Managed blank or null values using:

* Manual corrections
* IF functions
* Data validation techniques

---

## Step 4: Standardize Text Data

Corrected inconsistent text formatting.

### Example

Before:

```text
new york
NEW YORK
New york
```

After:

```text
New York
```

### Functions Used

```excel
=PROPER(A2)
=UPPER(A2)
=LOWER(A2)
```

---

## Step 5: Clean Extra Spaces

Removed unwanted spaces using:

```excel
=TRIM(A2)
```

---

## Step 6: Format Dates

Converted inconsistent date formats into a standard structure.

### Example

Before:

```text
12/5/22
2022-12-05
05-Dec-2022
```

After:

```text
05-Dec-2022
```

---

## Step 7: Data Validation

Validated data entries to prevent incorrect values.

### Examples

* Dropdown Lists
* Number Validation
* Date Validation
